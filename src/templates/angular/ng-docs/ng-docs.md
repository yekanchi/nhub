# Angular Tips 

## CLI:

* `ng generate component [component name]` Creates a new component
    * `ng create component product-item`
* `ng generate service`
    * `ng generate service userService`
* `ng add [package-name]` [adds the npm packages]
    * `ng add @angular/material`
* `ng serve`
* `ng build --prod`
* `ng test`
