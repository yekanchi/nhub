To use this FileDb LinqPad driver, do the following:

1.  In LinqPad, open the "Add Connection" dialog.
2.  At the bottom left, click the "View more drivers" button to open the "Choose a Driver" dialog.
3.  At the bottom left of this dialog, click the "Browse to a .lpx file" button.
4.  Using the "Open File" dialog, locate filedb_linqpad_driver.lpx file (where this Readme file is).
5.  That's it!  From now on the FileDb Driver will appear in the list of Dynamic Drivers in the "Add Connection" dialog.

The Northwind database files for FileDb are also included, along with some sample LINQ queries so you can begin using it immediately.
